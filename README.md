# Hangman Game

A simple command-line Hangman game in Python. Guess the letters of a hidden word before your attempts run out!

## Features

- Randomly selects a word from a predefined word bank.
- Shows your progress with underscores (`_`) for unguessed letters.
- Gives 10 attempts to guess the word.
- Notifies when you guess a letter correctly or incorrectly.
- Congratulates you when you guess the word, or reveals the word when you run out of attempts.

## Word Bank

Currently includes:
- rizz
- ohio
- sigma
- tiktok
- skibidi

You can easily add more words to `word_bank` in `hangman.py`.
